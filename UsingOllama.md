# Using Ollama

## Install Curl if its not there

```
sudo apt install curl
```

## Install Ollama

- ref : 
> https://ollama.com/download

```
curl -fsSL https://ollama.com/install.sh | sh
```

## Pull and run llama2 model

- list of available  models ::
> https://ollama.com/library

```
ollama run llama2
```

## Exit from Ollama

```
\bye
```
